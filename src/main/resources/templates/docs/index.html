Popular Music GET API

HTTP request

GET /api/music/popular?searchType=1&syy=2023&smm=04&eyy=2023&emm=04 HTTP/1.1
Content-Type: application/json;charset=UTF-8
Accept: application/json
Host: localhost:8080

Query parameters

Parameter Description

searchType

검색 타입 (1 = 가요 2 = POP 3 = J-POP)

syy

시작 연도(ex. 2023)

smm

시작 월(ex. 04)

eyy

종료 연도(ex. 2023)

emm

종료 월(ex. 04)

Music Search GET API

HTTP request

GET /api/music/search?searchType=1&searchKeyWord=%EC%95%84%EB%A6%AC%EB%9E%91&searchNation=KOR&page=0 HTTP/1.1
Content-Type: application/json;charset=UTF-8
Accept: application/json
Host: localhost:8080

Query parameters

Parameter Description

page

요청 페이지 정보

searchType

검색 타입 (1 = 제목, 2 = 가수 , 4 = 작사가 , 8 = 작곡가, 16 = 곡번호)

searchKeyWord

검색 키워드

searchNation

검색 국가
